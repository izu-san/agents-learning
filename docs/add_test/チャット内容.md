# チャット内容

サイトに以下の内容を追加した。

* 対象ページ: 会員登録、宿泊予約、マイページ
* 追加した項目: 性別、年齢

## Planner

* [`/specs/standard/e2e-plan.md`](../../specs/standard/e2e-plan.md) をコンテキストに追加した状態で

```
テスト対象のページに以下の項目が追加されました。

* 対象ページ: 会員登録、宿泊予約、マイページ
* 追加した項目: 性別、年齢

これらの追加された項目に対するテスト計画書を生成して追記してください。
追記した項目は明確にわかるようにしてください。
テスト観点は既存のテストを踏襲してください。
```

## Generator

* [`/specs/standard/e2e-plan.md`](../../specs/standard/e2e-plan.md) をコンテキストに追加した状態で

```
添付の `e2e-plan.md` を参照して、追加されたテストを実装してください。
実装方式は `/tests/standard/` ディレクトリに従い、成果物の出力先も同フォルダとします。
```

## Healer

```
tests/standard/ディレクトリに存在するテストを実行し、もしエラーが発生した場合は修正してください。
コードを修正した場合、修正内容などをコメントとしてコード内に残してください。
作成したテストの初回実行でまずは安定化が目的となります。
テスト内容を完全に無効化してしまうようなコード変更は禁止します。
```
